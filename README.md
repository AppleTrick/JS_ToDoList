# JS_ToDoList

Language : Javascript / HTML / CSS
IDE : VSCode

## 구현기능
- 배경화면 랜덤 변화
- 로그인기능 (localstorage 사용)
- Time 기능
- 위치, 날씨, 지역이름 표시 (geolocation 활용)
- 무작위 명언 구현하기
- To-Do-List 구현
